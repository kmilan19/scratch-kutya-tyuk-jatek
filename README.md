# Falusi Hajsza - Scratch Játék

Ez egy interaktív játék, amelyet Scratch-ben készítettem. 

## A játék célja
A játékos egy kutyát irányít a nyilakkal, és minél több tyúkot kell elkapnia 30 másodperc alatt.

## Megvalósított funkciók
* Két szereplő (Kutya és Tyúk) egyedi animációkkal.
* Változók: Pontszám és visszaszámláló időzítő.
* Véletlenszerű mozgás a tyúknál.
* Hangeffektusok ütközéskor.
